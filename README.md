> MMO game bot for Delta Chat


## Install

To install, open your terminal and run:

```sh
pip install deltaland
```

## Usage

Configure the bot:

```sh
deltaland init bot@example.com PASSWORD
```

Start the bot:

```sh
deltaland
```

Run `deltaland --help` to see all available options.
